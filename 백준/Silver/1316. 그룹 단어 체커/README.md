### 문제 설명

<p>그룹 단어란 단어에 존재하는 모든 문자에 대해서, 각 문자가 연속해서 나타나는 경우만을 말한다. 예를 들면, ccazzzzbb는 c, a, z, b가 모두 연속해서 나타나고, kin도 k, i, n이 연속해서 나타나기 때문에 그룹 단어이지만, aabbbccb는 b가 떨어져서 나타나기 때문에 그룹 단어가 아니다.</p>

<p>단어 N개를 입력으로 받아 그룹 단어의 개수를 출력하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 단어의 개수 N이 들어온다. N은 100보다 작거나 같은 자연수이다. 둘째 줄부터 N개의 줄에 단어가 들어온다. 단어는 알파벳 소문자로만 되어있고 중복되지 않으며, 길이는 최대 100이다.</p>

### 출력 

 <p>첫째 줄에 그룹 단어의 개수를 출력한다.</p>

### 알고리즘
<p>
 1. 단어의 개수 입력받기
 2. for 반복문을 단어 개수 만큼 실행하며, 단어를 1개씩 입력받기
 3. 입력된 단어가 '그룹 단어'인지 검사하는 함수 만들기
 4. '그룹 단어'가 맞으면, count 변수 1증가 시키기
 5. 최종 결과 출력하기
</p>

### 막혔던 부분
<p>
 그룹 단어가 아니라는 건, 어떤 문자가 연속 사용이 끝난 후에 나중에 다시 그 문자가
 등장하는 경우입니다. 이 기록을 어떻게 저장 할지 몰랐습니다.

 그러나 알파벳 26칸짜리 배열을 만들어서 값을 0과 1을 주며 각 알파벳의 등장여부를 저장해
 이 문제를 해결했습니다. if (cur_char != pre_char) 조건을 추가해서, 연속된 문자가 끝나고 
 새 문자 그룹이 시작될 때만 이 '방문 기록'을 검사하도록 만들었습니다.
 char는 아스키 코드라는 것을 고려하면 정수라서 'a'부터 'z'까지 연속된
 숫자로 저장되기 때문에 어떤 문자든 'a'를 빼면 0부터 시작하는 인덱스가 됩니다.

 이 index를 이용해서 배열을 검사했습니다. 
</p>
